# AppsDev Project

This repository contains the `coffee-shop` Angular application located in the `coffee-shop/` folder.

## Overview

- `coffee-shop/` - Angular application root
- `coffee-shop/src/` - Application source files
- `coffee-shop/public/` - Static assets and public files

## Getting Started

From the `coffee-shop/` folder:

```bash
cd coffee-shop
npm install
npm start
```

Then open the app in your browser at `http://localhost:4200`.

## Important Files

- `coffee-shop/package.json` - package dependencies and scripts
- `coffee-shop/angular.json` - Angular CLI workspace configuration
- `coffee-shop/src/app/` - application components, routes, and services
